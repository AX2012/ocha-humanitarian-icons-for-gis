In 2012, the United Nations Office for the Coordination of Humanitarian affairs (OCHA) has created a set of 500 freely available humanitarian icons to help relief workers present emergency and crisis-related information quickly and simply. These were created in png format. See Humanitarian and Country Icons 2012 for further details.

MapAction has converted these icons into a true type font format. These are then available as stylesheets for in ArcGIS and QGIS.
